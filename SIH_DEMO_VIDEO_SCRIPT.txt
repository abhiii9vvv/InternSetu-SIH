BACKGROUND:
The PM Internship Scheme receives applications from youth across India, including 
rural areas, tribal districts, urban slums, and remote colleges. Many of these 
candidates are first-generation learners with limited digital exposure and no 
prior internship experience. With hundreds of internships listed on the portal, 
it becomes difficult for such candidates to identify which ones match their 
skills, interests, or aspirations.

KEY CHALLENGES:
- Misaligned applications and missed opportunities
- Information overload for first-generation learners
- Limited digital literacy among rural candidates
- Need for mobile-compatible, simple interface
- Multi-language support requirement
- Integration with existing government portals

EXPECTED SOLUTION:
A functional prototype that:
‚Ä¢ Captures basic candidate inputs (education, skills, sector interests, location)
‚Ä¢ Uses a rule-based or ML-light model to suggest 3-5 top internships
‚Ä¢ Has a simple, intuitive UI with minimal text and visual cues
‚Ä¢ Can be accessed on mobile devices and adapted to regional language use
‚Ä¢ Outputs recommendations in a clear, user-friendly format

```typescript
calculateOverallScore(userProfile, internship) {
  const educationScore = this.calculateEducationMatch(userProfile.education, internship.requiredEducation);
  const skillsScore = this.calculateSkillsMatch(userProfile.skills, internship.requiredSkills);
  const locationScore = this.calculateLocationMatch(userProfile.location, internship.location);
  const interestScore = this.calculateInterestMatch(userProfile.interests, internship.sector);
  const preferenceScore = this.calculatePreferenceMatch(userProfile, internship);

  // Transparent weighted scoring
  const weightedScore = (
    educationScore * 25 +    // Government compliance
    skillsScore * 30 +       // Job performance
    locationScore * 20 +     // Feasibility
    interestScore * 15 +     // Career fit
    preferenceScore * 10     // Personal preferences
  ) / 100;

  return Math.round(weightedScore * 100) / 100;
}
```
